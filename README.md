In this repository you will find a code for AES algorithm made to encrypt and decrypt a file of your choice. 

Example of use: 

  You can create a .txt file, with something written there, or you can use "fisier1" as a test. Than open Console App and run "AES_test.exe <enc|dec> <input file> <output file>". You can choose the type of file you want at output to be encrypted/decrypted.
